(static) this.getInformation(deviceInfo)

Returns information about the current device such as its name, model, system version
Parameters:
Name Type Description
deviceInfo function Javascript function that will be called with the detailed device information. The format of the function is: deviceInfo(info). Available fields are: - info.name(string) - devices user name - info.serial(string) - unique device serial number - info.systemName(string) - devices system name - info.systemVersion(string) - firmware version - info.model(string) - model string

Example
function DeviceInfo(info)
{
    // Loop through the information and display it
    var infoStr="";
    for(var i in info)
    {
        infoStr+=i+": "+info[i]+"\n";
    }

    alert(infoStr)
}
